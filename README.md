# UCR-Time-Series-classification
To find the accuracy of the the UCR time series datasets, using kNN algorithm.

This repository has the files which can be used to find the error rate or accuracy of the datasets available on http://timeseriesclassification.com/index.php.
You can directly run the ED_DTW.py file and select the dataset's path with the window size (DTW) and it will show the results.
